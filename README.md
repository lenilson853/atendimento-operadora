# 🤖 Atendimento Virtual - Script Python

Este é um programa simples de atendimento virtual em Python que simula um menu interativo para consultar saldo, fazer recargas, ver promoções, contratar pacotes de internet e outros serviços.

## 🚀 Funcionalidades

- Consulta de saldo atual  
- Recargas em valores pré-definidos (R$10, R$20, R$30, R$40)  
- Ativação de promoções exclusivas  
- Contratação de pacotes de internet  
- Ativação de roaming e atendimento com atendente  
- Encerramento do atendimento  

## 💻 Tecnologias Utilizadas

- Python 3  
- Biblioteca `time` para simular tempo de espera e melhorar experiência do usuário  

## 🎯 Objetivo do Projeto

Este projeto tem como objetivo praticar:

- Estruturas condicionais (`if`, `elif`, `else`)  
- Laços de repetição (`while`)  
- Manipulação de variáveis e entrada de dados (`input()`)  
- Formatação de saída com `print()`  
- Uso básico de bibliotecas padrão (como `time`)  

## 📌 Como Executar

1. Certifique-se de ter o Python 3 instalado no seu computador.  
2. Salve o arquivo `atendimento_virtual.py` na sua máquina.  
3. Abra o terminal (Prompt de Comando) e navegue até a pasta onde está o arquivo.  
4. Execute o comando:

```bash
python atendimento_virtual.py
